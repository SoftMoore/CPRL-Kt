emitBranch

open override fun emitBranch(condition: Boolean, label: String)
Content copied to clipboard

For Boolean expressions, the method emits the appropriate branch opcode based on the condition. For example, if the expression is a "<" relational expression and the condition is false, then the opcode "BGE" is emitted. The method defined in class Expression works correctly for Boolean constants, Boolean named values, and "not" expressions. It should be overridden for other Boolean expression ASTs (e.g., RelationalExpr).

Parameters

condition

the condition that determines the branch to be emitted.

label

the label for the branch destination.

Throws

CodeGenException

if the method is unable to generate appropriate target code.
