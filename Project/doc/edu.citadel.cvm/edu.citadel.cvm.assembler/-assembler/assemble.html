assemble

fun assemble()
Content copied to clipboard

Assembles the source file. If there are no errors in the source file, the object code is placed in a file with the same base file name as the source file but with a ".obj" suffix.

Throws

java.io.IOException

if there are problems reading the source file or writing to the target file.
