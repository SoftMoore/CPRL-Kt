fun parseVariableExpr(): Variable
Content copied to clipboard

Parse the following grammar rule: variable = ( varId | paramId ) ( "[" expression "]" )* .

This method provides common logic for methods parseVariable() and parseNamedValue(). The method does not handle any parser exceptions but throws them back to the calling method where they can be handled appropriately.

Return

the parsed variable.

See also

.parseNamedValue

Throws

ParserException

if parsing fails.
